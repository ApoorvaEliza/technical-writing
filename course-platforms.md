# Education and course publishing platforms

These section consists of platforms that pay authors/technical writers to write technical courses.

## Educative

Educative hosts and provides interactive courses for software engineers. [Become an educative author](https://learn.educative.io/become-an-educative-author).

## Guru99

Guru99 as a platform provides free training tutorials and videos on IT courses. [Become an instructor](https://www.guru99.com/become-an-instructor).
